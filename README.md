# MoleMap --- Молекулярный анализ

**MoleMap** --- это веб-приложение для предсказания свойств молекул на
основе их химической структуры (SMILES).
Приложение использует современные ML-модели для вычисления ключевых
ADME/токсикологических параметров и предоставляет интерпретируемую
визуализацию через 2D и 3D molecular viewers.

## 🚀 Основные возможности

### 🔬 Анализ молекул

-   Ввод SMILES-нотации.
-   Автоматический расчёт ключевых параметров:
    -   **Растворимость**
    -   **Липофильность (logP)**
    -   **Кардиотоксичность**
    -   **Общая токсичность**
    -   **FDA одобрение (вероятность)**
    -   **CLINTOX** (риск токсичности)

### 🌡 Тепловая карта влияния атомов

2D-визуализатор показывает вклад каждого атома в выбранное свойство:
- **Оранжевые оттенки** --- увеличивают значение свойства
- **Зелёные оттенки** --- уменьшают значение

Кнопки внизу позволяют переключать режим подсветки:
`Растворимость`, `Липофильность`, `Кардиотоксичность`, `Токсичность`,
`FDA одобрение`.

### 🧬 3D Viewer

-   Объёмная визуализация структуры молекулы.
-   Возможность вращения, масштабирования, изучения геометрии.

### 💾 Сохранение в базу данных

Если результаты подходят пользователю, можно сохранить молекулу в базу
одним кликом --- кнопка **"Сохранить в БД"**.

## 🧠 Контекст и назначение приложения

Проект создан для исследователей, студентов химических специальностей и
разработчиков лекарств, которым необходимо быстро оценивать свойства
молекул без проведения дорогих и длительных экспериментов.

### ✨ Проблема, которую решает MoleMap

-   Химические, биологические и физические методы --- дорогие, медленные
    и часто небезопасные.
-   Исследователям требуется быстрый инструмент предсказания
    ADME/токсичности.

### ✔️ Наше решение

-   Мгновенное предсказание свойств на основе ML.
-   Интерпретируемые результаты.
-   Удобный визуальный интерфейс.

## 🏗 Архитектура

-   **ML-модули** --- модели ADME/токсичности.
-   **Облачная инфраструктура**.
-   **Frontend**: Vue.js + RDKit.js.
-   **Backend**: FastAPI + RDKit + ML-модели.
